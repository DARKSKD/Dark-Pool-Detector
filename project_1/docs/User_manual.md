# User Manual – Dark Pool Surveillance Dashboard

1. Start the dashboard using:
   streamlit run frontend/dashboard.py

2. Enable “Run Live Surveillance” toggle.

3. Observe:
   - Left panel: live trades
   - Right panel: alerts generated by detection engine

4. Select symbol from dropdown to investigate:
   - Trade timeline
   - Buy/Sell summary
   - Price impact

5. Stop system anytime using CTRL+C.
